# langscan
Extensão para identificiar palavras em outros idiomas

Chrome Extension – Text Extractor

Uma extensão para Google Chrome que captura automaticamente todo o texto presente na página atual e o disponibiliza para o usuário de forma simples e rápida.

🚀 Funcionalidades

🔍 Identifica todo o texto da página ativa

📄 Extrai conteúdos visíveis automaticamente

⚡ Execução rápida e leve

🧠 Pode ser usada para análise, estudos ou automações

📦 Instalação
🔧 Modo Desenvolvedor (Manual)

Baixe ou clone este repositório:

git clone https://github.com/SEU-USUARIO/NOME-REPO.git

Abra o Google Chrome e acesse:

chrome://extensions/

Ative o Modo do desenvolvedor (canto superior direito)

Clique em "Carregar sem compactação"

Selecione a pasta do projeto

🛠️ Como usar

Acesse qualquer site no navegador

Clique no ícone da extensão

A extensão irá:

Ler o conteúdo da página

Extrair todo o texto disponível

O resultado será exibido ou processado conforme a implementação
